# 🌱 Smart Irrigation AI System

## 📌 Overview

An AI-powered irrigation system that helps farmers optimize water usage by predicting soil moisture and making smart irrigation decisions.

## 🚀 Features

* Simulated IoT sensor data (moisture, temperature, humidity)
* Irrigation decision system
* Irrigation duration optimization
* Machine Learning prediction
* Interactive dashboard using Streamlit
* Alerts and insights

## 🧠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Output

* Real-time sensor data
* Irrigation decision
* Moisture prediction
* Graphs and insights

## 🎯 Goal

Reduce water waste and improve farming efficiency using AI.
